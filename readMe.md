# Buy&Dash
## `...From the comfort of your couch.`
## MERN Stack Application
### Status: (Active)

        Buy&Dash is an e-commerce application, 
        where the goods you buy is delivered to your door-step.

## Usage:

        1. Create Account.
        2. Log in.
        3. Search through items.
        4. Select item.
        5.Add to cart.
        6. View cart.
        7. Delete from cart.
        8. View orders.
        9. Checkout.
        10. Log out.
